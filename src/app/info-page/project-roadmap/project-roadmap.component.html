The project is conditionally divided into three stages:

The first stage is the most peaceful! The only function available filling in the tiles. Anyone may fill in any crayon he likes! The first stage ends after 999 tiles have been filled (including repeated fills of the same tiles).

After the 999th invocation of the Fill Tile function, the project enters the second stage. Users have the opportunity to buy the ownership of the tiles. This is not a complete project yet, but now only you can modify purchased by you.

While the number of modified tiles (including re changes) reaches 4999 and the number of purchased tiles reaches reaches 4999 the second stage will continue.

The third stage begins when both conditions are met - the number of filled The number of tiles reaches 4999 and the number of purchased tiles reaches 499. At this point the project starts in full force! Tiles can buy, transfer, put up for sale, sold through the auction, rent them out. Game on!
